Gestión Inteligente de Cuentas

Sistema web local para administrar operaciones bancarias según el estado actual de una cuenta.

Estado inicial Sin cuenta La simulación comienza al crear una cuenta bancaria.

Contexto

Una cuenta bancaria puede comportarse de forma diferente dependiendo de si está activa, bloqueada, sobregirada o cerrada.

Problema

Manejar muchas reglas con condicionales vuelve el código más difícil de mantener y de ampliar.

Solución

Separar el comportamiento por estado permite que el sistema sea más claro, flexible y ordenado.

Flujo de estados

Comportamiento general de la cuenta durante la simulación.

Activa → Sobregirada → Activa
Activa → Bloqueada → Activa / Sobregirada
Activa / Bloqueada → Cerrada
