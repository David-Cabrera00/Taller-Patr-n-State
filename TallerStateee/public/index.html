Operaciones bancarias

Cada acción se comporta diferente según el estado de la cuenta.

Reglas por estado

Así se evidencia el comportamiento polimórfico del patrón State.

Activa Deposita, retira, transfiere, bloquea y puede cerrar si saldo es cero.
Sobregirada Solo permite depositar para pagar el sobregiro.
Bloqueada Permite depósitos y desbloqueo. No permite retiros ni transferencias.
Cerrada No permite ninguna operación bancaria.

Transiciones del patrón State

El objeto cuenta cambia de comportamiento sin usar condicionales complejas.

Activa → Sobregirada → Activa | Bloqueada → Activa / Sobregirada | Cerrada
